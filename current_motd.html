Chat commands

All
  • /addq [url_1] [url_2] ... - adds urls to the playlist
  • /img [url] - posts image/gif
  • /date - stream time in the local timezone
  • /date2 - stream time2 in the local timezone
Admin
  • /autostart - toggle autostart on countdown = 0
  • /autobg [url] - saves background image for autostart
  • /bgset [url] - changes background image to autobg
  • /editbg [url] - changes background image
  • /editbanner [url] - changes banner
  • /purge - delete playlist
  • /clear - clears chat
  • /cdlocal [year] [month] [day] [hour] [minute] - set countdown date (local)
  • /cdlocal2 [year] [month] [day] [hour] [minute] - set countdown2 date (local)
  • /addemote [tag] [url]
  • /skip - remove current video
