Chat commands

All (system)
  • /addq [url] - adds urls to the playlist
  • /img [url] - posts image/gif
  • /pick [membername] - display the member's picks(?)
  • /rule[1-8]
  • /voteskip - vote to skip this video
  • /skipclear - remove voteskip counter
  • /skiphide - do not display voteskip counter
  • /skipshow - display voteskip counter
All (chat fluff)
All (chat fluff sound)
Mod
  • /purge - delete playlist
  • /clear - clears chat
  • /addemote [tag] [url]
  • /msgdel [true/false]
  • /lock - toggle playlist lock
Admin (system)
  • /editbg [url] - changes background image
  • /editbanner [url] - changes banner
  • /muteall - mute chat
  • /unmuteall - unmute chat
  • /maxq [time] - changes maximum queue time
  • /maxv [number] - changes maximum number of video
  • /voteratio [ratio] - changes voteskip ratio
  • /vs[on/off] - turn on voteskip
  • /noise[on/off] - turn on emote notice
  • /chatimg [url/false] - set chatimg or remove chat img (set false)
  • /chatimgop [opacity(0-1)] - change chatimg opacity
  • /chatlimit[on/off] - limit chat messages to x seconds
  • /setchatlimit [seconds] - set chat message delay in seconds
  • /amqq- embed amq to the playlist
Admin (screen fluff)
  • !event1 - will loop until stopped
  • /stope1 - cancel event1
  • /event1bg [image url] - set event1 bg
  • /event1music [audio url] - set event1 music
  • /event1img [image url] - set event1 img
  • /event1effect [image url] - set event1 effect
  • /event1effectedit - enable edit effect1 position
  • !img1 - shows img1 on the screen
  • /img1 [image/gif url] - set img1
  • /img1hide - hide img1
  • /img1edit - enable edit
  • !img1fixed - shows fixed img1 on the screen

Misc.

  • Name color change: add color hex code to your profile text
  • "Admin" commands require permission to change js and css
